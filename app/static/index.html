Cómo usar la API Key

          
Incluye la cabecera X-API-Key en tus peticiones HTTP. No es necesario JWT ni X-BU-ID — la clave ya lleva el contexto de BU.

          
POST /assessments/{assessment_id}/run
X-API-Key: cnt_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
Content-Type: application/json

{
  "document_id": "uuid-del-documento"
}
